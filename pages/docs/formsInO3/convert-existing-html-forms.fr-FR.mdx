# Comment convertir les formulaires HTML existants 

De nombreux utilisateurs ont déjà utilisé le module de saisie de formulaires HTML d'OpenMRS et ont des ensembles de formulaires existants qu'ils souhaitent convertir en formulaires O3. 

Heureusement, vous n'avez pas à repartir de zéro, grâce à l'outil Automated Schema Migration créé par Palladium-Kenya : https://github.com/palladiumkenya/openmrs-module-afyastat/blob/1519b9e0b6a7d6b0ac8ac9e1cfe4cbab5c608aa8/README2.md#automation-of-html-to-json-schema-migration 

Cet outil prend vos schémas HFE existants et les convertit en un schéma JSON qu'O3 peut exploiter. Il ne pourra probablement pas convertir 100% de vos schémas HFE, mais il vous permettra d'aller assez loin pour que les ajustements restants puissent être faits soit dans le JSON, soit avec l'outil GUI Form Builder. 

Plus d'informations ici sur le [OpenMRS Talk Forum] (https://talk.openmrs.org/t/o3-new-initiative-to-automate-hfe-o3-forms/36825).
