# Configurez votre premier widget à l'aide d'un modèle simple

Pour ce faire, il suffit d'ajouter une extension à l'emplacement patient-chart-allergies-dashboard de l'application esm-patient-chart-app. 

Remarque : la "couleur" intervient dans l'aspect visuel du graphique. Le code couleur que vous choisissez dicte la couleur de la ligne de données dans le graphique.

### Exemple de configuration
```json
{
  "@openmrs/esm-patient-chart-app": {
    "extensionSlots": {
      "patient-chart-allergies-dashboard-slot": {
        "add": [
          "obs-by-encounter-widget#hiv",
          "obs-by-encounter-widget#diabetes"
        ],
        "configure": {
          "obs-by-encounter-widget#hiv": {
            "title": "HIV Widget",
            "resultName": "HIV",
            "data": [
              {
                "concept": "1132AAAAAAAAAAAAAAAAAAAAAAAAAAAAA",
                "label": "sodium",
                "color": "green"
              },
              {
                "concept": "856AAAAAAAAAAAAAAAAAAAAAAAAAAAAA",
                "label": "Viral load",
                "color": "red"
              }
            ],
            "table": {
              "pageSize": 5
            }
          }
        }
      }
    }
  }
}
```
