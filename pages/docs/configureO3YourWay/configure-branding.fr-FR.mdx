# Configurer votre image de marque

## Ajouter votre logo

Pour l'écran de connexion: dans l'application openmrs/esm-login-app (exemple de configuration ci-dessous)
Pour la bannière: dans l'application openmrs/esm-primary-navigation-app (exemple de configuration ci-dessous)

## Changer les couleurs

Ceci peut être configuré via le module openmrs/esm-styleguide avec ces éléments de configuration. Exemple de configuration:

```
{
  "@openmrs/esm-styleguide": {
    "Couleur de la marque #1": "#00473F",
    "Couleur de la marque #2": "#363463",
    "Couleur de la marque #3": "#51A351"
  }
}
```

### Légende de l'impact des couleurs primaires, secondaires et tertiaires:

- La couleur de la marque #1:\_ (couleur primaire) a un impact sur les éléments suivants:

  - Couleur de la barre de navigation globale

  ![Couleur
de survol](/global-nav-bar-color.png)

  ![Siderail Highlight Circle color](/hover-color.png)

- Couleur de la marque #2:\_ (Couleur secondaire) A un impact sur les éléments suivants:

  - Couleur non sélectionnée de l'App Nav

- Couleur de la marque #3:\_ (Couleur tertiaire) A un impact sur les éléments suivants:

  - Titre du widget Couleur de soulignement

  - Couleur du bouton (non sélectionné)

## Exemple de fichier config.json

Vous pouvez configurer votre image de marque de plusieurs façons. En fin de compte, cela se trouve dans votre fichier config.json - voir [cet exemple ici](https://github.com/UCSF-IGHS/openmrs-esm-ohri/blob/master/packages/esm-ohri-core-app/src/ohri-patient-dashboards-config.json) sur GitHub ou voir l'exemple ci-dessous:

```json copy
{
  "@openmrs/esm-patient-chart-app": {
    "extensionSlots": {
      "patient-chart-dashboard-slot": {
        "remove": [
          "test-results-summary-dashboard",
          "conditions-summary-dashboard",
          "results-summary-dashboard",
          "programs-summary-dashboard",
          "allergies-summary-dashboard",
          "attachments-results-summary-dashboard",
          "drug-orders-summary-dashboard"
        ]
      }
    },
    "notesConceptUuids": [
      "161011AAAAAAAAAAAAAAAAAAAAAAAAAAAAAA",
      "437d1e25-e7ab-481c-aabc-01f21c6cdef1"
    ]
  },
  "@openmrs/esm-patient-banner-app": {
    "extensionSlots": {
      "patient-actions-slot": {
        "remove": ["add-patient-to-patient-list-button"]
      }
    }
  },
  "@openmrs/esm-styleguide": {
    "Brand color #1": "#1A3054",
    "Brand color #2": "#131E35",
    "Brand color #3": "#0078A6"
  },
  "@openmrs/esm-login-app": {
    "logo": {
      "src": "https://ohri-demo.globalhealthapp.net/openmrs/images/logos/ohri_logo_lightbg.svg",
      "alt": "Logo"
    }
  },
  "@openmrs/esm-primary-navigation-app": {
    "logo": {
      "src": "https://ohri-demo.globalhealthapp.net/openmrs/images/logos/ohri_logo_darkbg.svg"
    }
  }
}
```
