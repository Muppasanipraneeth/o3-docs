# Personnaliser votre carte d'importation

Une carte d'importation est un fichier JSON qui mappe les noms de modules aux URL. Dans ce cas, ces modules sont ce qu'on appelle des [modules frontend](/docs/frontend-modules/overview) ou microfrontends dans O3. L'URL est l'URL du fichier `index.js` du module. L'URL peut être un chemin relatif ou une URL absolue. Dans ce cas, les URL sont relatives à l'emplacement du fichier de carte d'importation sur le serveur où le frontend est hébergé.

Les implémenteurs ont la possibilité de choisir quels modules frontend inclure dans leur instance. Cela se fait en personnalisant la carte d'importation. Vous pouvez ajouter ou supprimer des URL de module frontend du fichier `spa-build config` puis reconstruire l'application de référence de la distribution. La carte d'importation sera mise à jour en conséquence.
